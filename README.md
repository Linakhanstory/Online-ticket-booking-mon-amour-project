Mon Amour Grand is a luxury event booking platform designed for high-end ballet performances and fine art gallery shows. It provides a seamless, elegant experience for users to discover elite cultural events and manage their bookings digitally.

Project Description:
This is a full-stack web application that brings a "luxury concierge" feel to the digital space.  Purpose: To provide an elite booking interface for high-end arts and performances.  Architecture: Built using a modern stack featuring a vanilla JavaScript frontend and a Node.js/Express backend.  
Key Features: 
Includes secure user authentication, a dynamic event gallery, and a functional shopping cart for ticket management.  

Project Output: 
The final product is a live, interactive booking ecosystem:  Frontend: A responsive, high-end UI featuring smooth animations and a "Champagne and Obsidian" aesthetic.  User System: A working login/sign-up flow that stores user sessions locally.  Booking Engine: An interactive ticket selector where users can choose between Standard, Premium, and VIP tiers. 

Database Integration: 
A persistent MongoDB backend that saves every transaction and booking detail automatically.

Tech Stack
Frontend: HTML5, Tailwind CSS, Vanilla JavaScript  

Backend: Node.js, Express.js  

Database: MongoDB, Mongoose  

Tools: Fetch API, LocalStorage for session management
